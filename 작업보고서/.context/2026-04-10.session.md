### Console 매핑
| 보드 | Zephyr console | TX 핀 | RX 핀 | host device | baud |
|------|----------------|-------|-------|-------------|------|
| Tower | `&uart0` | P0.20 | P0.19 | /dev/ttyUSB1 | 115200 |
| Link  | `&uart1` | P0.16 | P0.15 | /dev/ttyUSB0 | 115200 |

### USB CDC 매핑
- ttyACM0~1: J-Link S/N 1050295470 (Tower)
- ttyACM2~3: J-Link S/N 1050234191 (Link)
- **ttyACM4**: REVITA TOWER USB CDC (VID 0x2FE3, PID 0x0100)

### RS485 센서 검증 결과
| 센서 | Slave ID | Baud rate | 프로토콜 |
|------|----------|-----------|----------|
| 토양센서 #1 | 0x01 | **4800** | Modbus RTU FC=0x03, reg 0x0000, cnt 2 |
| 토양센서 #2 | 0x01 | **9600** | 동일 |
| 엽온센서 | 0x02 | **9600** | 동일 |
- 변환: raw / 10.0 → %, °C
- CRC16-Modbus: poly 0xA001, init 0xFFFF, LSB first

### Motor Driver 핀 (Link)
| 채널 | EN_A | EN_B | EN_P2 (LOW 고정) | 12V_EN |
|------|------|------|-------------------|--------|
| X | P0.14 | P0.13 | P0.04 | P0.17 |
| Y | P0.25 | P1.01 | P1.02 | P0.17 |
- **EN_P2는 반드시 LOW** — motor 동작과 무관
- CW: EN_A=HIGH, EN_B=LOW / CCW: EN_A=LOW, EN_B=HIGH

### Host 시뮬레이션
- **웹 서버**: Flask @ port 8080, http://192.168.0.29:8080
- **시리얼**: /dev/ttyACM4, 115200 baud
- **프로토콜**: 텍스트 라인 ('\n' terminated)
  - TX: CMD:LED_ON, CMD:STATUS, DATA:xxx
  - RX: ACK:xxx, SENSOR:temp=22.5,humi=55.0, STATUS:uptime=N

### 디버깅 이력 (해결됨)
1. **RAK4630 교체 시 J-Link 접속 실패**: SWD 접촉 불량 → 다른 모듈로 교체
2. **Node LED P0.15 안 깜빡임**: UART RX가 P0.15 점유 → RX를 P0.19로 이동
3. **토양센서 9600에서 응답 없음**: baud rate 4800이었음 → 4800으로 변경 후 정상
4. **Flask 포트 5000 외부 접속 불가**: 포트 8080으로 변경 시 정상

printf "r\ng\nexit\n" | /opt/SEGGER/JLink_V924a/JLinkExe -nogui 1 -if swd -speed 4000 -device nRF52840_xxAA -SelectEmuBySN 1050295470

# Link 리셋
printf "r\ng\nexit\n" | /opt/SEGGER/JLink_V924a/JLinkExe -nogui 1 -if swd -speed 4000 -device nRF52840_xxAA -SelectEmuBySN 1050234191

# Host 웹 시뮬레이션 실행
cd /home/uttec/revita/zephyr_workspace/system/driver/host_test/web
python3 app.py /dev/ttyACM4  # 포트 8080

# ttyACM 디바이스 확인
for d in /dev/ttyACM*; do echo "=== $d ==="; udevadm info -q property "$d" | grep -E "ID_MODEL|ID_SERIAL"; done
