### 2.2 환경 설치 명령어

```bash
# 1. 의존성 설치
sudo apt update
sudo apt install --no-install-recommends \
    git cmake ninja-build gperf ccache dfu-util \
    device-tree-compiler wget python3-dev python3-pip \
    python3-setuptools python3-tk python3-wheel \
    xz-utils file make gcc gcc-multilib g++-multilib \
    libsdl2-dev libmagic1

# 2. west 설치
pip3 install --user west

# 3. Zephyr 소스 가져오기
cd ~
west init -m https://github.com/zephyrproject-rtos/zephyr --mr v3.6.0 zephyrproject
cd zephyrproject
west update

# 4. Python 의존성 설치
pip3 install --user -r ~/zephyrproject/zephyr/scripts/requirements.txt

# 5. Zephyr SDK 설치
cd ~
wget https://github.com/zephyrproject-rtos/sdk-ng/releases/download/v0.16.5/zephyr-sdk-0.16.5_linux-x86_64.tar.xz
tar xvf zephyr-sdk-0.16.5_linux-x86_64.tar.xz
cd zephyr-sdk-0.16.5
./setup.sh

# 6. 환경 변수 설정 (~/.bashrc에 추가)
export ZEPHYR_BASE=~/zephyrproject/zephyr
export PATH=~/.local/bin:$PATH
source $ZEPHYR_BASE/zephyr-env.sh
```

## 3. Phase 1: BLE 통신 구현

### 3.1 목표

```
Target2 (Central/TX) ──BLE──> Target1 (Peripheral/RX)
     │                              │
     │  Connection + GATT Write     │
     │  카운터 데이터 전송          │
     └──────────────────────────────┘
```

### 3.2 BLE 아키텍처

```
┌─────────────────────────────────────────────────────────┐
│  BLE Peripheral (Target1 - RX)                          │
├─────────────────────────────────────────────────────────┤
│  • Advertising: "REVITA_LINK"                           │
│  • GATT Server                                          │
│    └── Custom Service (UUID: 12345678-1234-...)        │
│        ├── RX Characteristic (Write)                   │
│        └── TX Characteristic (Notify)                  │
│  • Connection 수신 대기                                 │
│  • 데이터 수신 시 UART 출력                             │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  BLE Central (Target2 - TX)                             │
├─────────────────────────────────────────────────────────┤
│  • Scan: "REVITA_LINK" 검색                             │
│  • GATT Client                                          │
│  • Connection 요청                                      │
│  • 주기적 데이터 전송 (카운터)                          │
└─────────────────────────────────────────────────────────┘
```

### 3.3 Custom GATT Service

```
REVITA Service
├── UUID: 12345678-1234-5678-1234-56789abcdef0
│
├── RX Characteristic (Central → Peripheral)
│   ├── UUID: 12345678-1234-5678-1234-56789abcdef1
│   ├── Properties: Write, Write Without Response
│   └── 용도: Central이 데이터 전송
│
└── TX Characteristic (Peripheral → Central)
    ├── UUID: 12345678-1234-5678-1234-56789abcdef2
    ├── Properties: Notify
    └── 용도: Peripheral이 데이터 전송
```

```
Zephyr LoRaWAN Stack:
┌─────────────────────────────────────────────────────────┐
│  Application                                            │

│  └── Region: KR920                                     │
├─────────────────────────────────────────────────────────┤
│  LoRa PHY Driver                                        │
│  └── SX126x Driver                                     │
├─────────────────────────────────────────────────────────┤
│  SPI Driver                                             │
└─────────────────────────────────────────────────────────┘
```

### 5.2 RAK4630 핀 맵 (Device Tree)

```dts
/* nrf52840 → RAK4630 (SX1262) */
&spi1 {
    compatible = "nordic,nrf-spim";
    status = "okay";
    pinctrl-0 = <&spi1_default>;
    cs-gpios = <&gpio1 10 GPIO_ACTIVE_LOW>;  /* NSS: P1.10 */

    sx1262: sx1262@0 {
        compatible = "semtech,sx1262";
        reg = <0>;
        spi-max-frequency = <8000000>;
        reset-gpios = <&gpio1 6 GPIO_ACTIVE_LOW>;   /* P1.06 */
        busy-gpios = <&gpio1 14 GPIO_ACTIVE_HIGH>;  /* P1.14 */
        dio1-gpios = <&gpio1 15 GPIO_ACTIVE_HIGH>;  /* P1.15 */
        antenna-enable-gpios = <&gpio1 5 GPIO_ACTIVE_HIGH>; /* P1.05 */
    };
};
```

## 8. 빌드 및 플래시 명령어

### 8.1 BLE Peripheral (Target1)

```bash
# 빌드
cd ~/zephyrproject
west build -b nrf52840dk_nrf52840 \
    /home/uttec/revita/zephyr_workspace/revita_ble_peripheral \
    -d build_peripheral

# 플래시 (Target1: SN 001050234191)
west flash -d build_peripheral --snr 001050234191

# RTT 로그 확인
JLinkRTTClient
```
