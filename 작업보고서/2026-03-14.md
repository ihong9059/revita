### 3. MCP23017 GPA 문제 분석

#### 증상
- GPA 3개 이상 동시 HIGH 시 MCP23017 리셋 발생
- IODIRA가 0xFF로 초기화됨 (출력→입력으로 변경)

#### 원인
- GPA 출력에 Si2302CDS N-channel MOSFET 게이트 연결
- 다수 MOSFET 동시 구동 시 전류 과부하
- Si2302CDS 특성: VDS=20V, ID=2.6-3.5A, RDS(on)=57-75mΩ, Vth=0.5-1.2V

#### 테스트 결과
| 테스트 | 결과 |
|--------|------|
| GPA 1-2핀 동시 ON | ✅ 정상 |
| GPA 3핀 이상 동시 ON | ❌ 리셋 발생 |
| GPB 전체 ON | ✅ 정상 |

#### 해결 방안
- 하드웨어: 게이트 저항 추가 또는 전류 제한
- 소프트웨어: 동시 출력 핀 수 제한

### 5. nRF52840 2.4GHz Radio 통신 테스트

#### 설정 (Nordic Proprietary 1Mbps)
| 파라미터 | 값 | 설명 |
|----------|-----|------|
| 모드 | NRF_1MBIT | Nordic 1Mbps |
| 주파수 | 2480 MHz | 채널 80 |
| TX Power | +8 dBm | 최대 출력 |
| 주소 | 0xE7E7E7E7 | ShockBurst 표준 |
| BALEN | 3 | 3바이트 기본 주소 |
| CRC | 2 bytes | CRC-16-CCITT |

#### 패킷 형식
```
[LEN=04][0xAA][0xBB][counter][0xDD]
```
- LEN: 페이로드 길이 (4바이트)
- counter: 0x00~0xFF 순환

#### 통신 테스트 결과
| 항목 | 값 | 상태 |
|------|-----|------|
| RSSI | -43 dBm | 매우 좋음 |
| CRC 검증 | 활성화 | 노이즈 필터링 |
| 패킷 손실 | 0% | 안정적 |
| 수신 간격 | ~500ms | 정상 |

#### RSSI 품질 기준
| RSSI 범위 | 품질 |
|-----------|------|
| 0 ~ -30 dBm | 최상 |
| -30 ~ -50 dBm | 매우 좋음 ← 현재 |
| -50 ~ -70 dBm | 좋음 |
| -70 ~ -85 dBm | 보통 |
| -85 ~ -100 dBm | 약함 |

#### 해결된 이슈
1. **주소 설정 오류**: BASE0/PREFIX0 레지스터 설정 수정
2. **RSSI 측정**: SHORTS 레지스터로 자동 RSSI 샘플링 (ADDRESS_RSSISTART)
3. **노이즈 필터링**: CRC 활성화로 무효 패킷 제거
