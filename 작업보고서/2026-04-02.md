### 3. 주요 작업 내용

#### 3.1 Auto Refresh 갱신 알림 추가

- Web Audio API `beep()` 함수 추가 (800Hz, 100ms)
- `AudioContext` 재사용 + `resume()` 호출로 브라우저 자동재생 정책 대응
- Mac Safari/Chrome에서 소리 재생 안 되는 문제 발생 → 서버 미재시작이 원인
- 대안으로 **화면 플래시 알림** 추가: 화면 중앙에 흰색 "DATA RECEIVED" 표시, 1초 후 사라짐
- Auto Refresh(5초), LoRa Auto Refresh(2초) 모두에 적용

#### 3.2 버전 표시 추가

- 타이틀 바에 `v1.x` 표시 추가
- 코드 수정 후 브라우저 캐시 문제 해결을 위한 버전 확인 용도

#### 3.3 Flask 서버 재시작 이슈

- 코드 수정 후 서버 재시작 필요 (Flask debug=False)
- 이전 세션(4월1일)의 서버 프로세스가 계속 실행 중이어서 수정사항 미반영
- `kill` + 재시작 + 브라우저 `Cmd+Shift+R` 강제 새로고침으로 해결

#### 3.4 Self Test 기능 현황 확인

- Web UI: `0x01F1`(NO-RF), `0x01F2`(RF 포함) 전체 Self Test 버튼 존재
- 펌웨어: 해당 레지스터 핸들러 미구현
- Flash Self Test(`0x0150`, value=2)만 구현되어 동작
