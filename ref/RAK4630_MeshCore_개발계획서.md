# RAK4630 MeshCore LoRa Mesh 개발 계획서

## 1. 프로젝트 개요

### 1.1 프로젝트명
**RAK4630 기반 MeshCore LoRa Mesh 네트워크 구축**

### 1.2 목적
RAK4630 모듈에 MeshCore 펌웨어를 설치하여 인터넷 독립적인 LoRa Mesh 통신 네트워크를 구축한다.

### 1.3 배경
- 오프그리드 환경에서의 통신 필요성
- 재난/긴급 상황 대비 통신 인프라 구축
- IoT 센서 네트워크 확장

## 4. 펌웨어 설치

### 4.1 방법 1: 웹 플래셔 사용 (권장)

#### 절차
1. Chrome 또는 Edge 브라우저 열기
2. https://flasher.meshcore.co.uk 접속
3. RAK4630 USB 연결
4. 장치 목록에서 "RAK4630" 또는 "RAK4631" 선택
5. 펌웨어 유형 선택:
   - **Companion Radio**: 스마트폰 앱 연동용
   - **Repeater**: 리피터/중계기용
   - **Room Server**: 채팅 서버용
6. "Flash" 버튼 클릭
7. 시리얼 포트 선택 (팝업)
8. 플래싱 완료 대기

### 4.2 방법 2: PlatformIO 빌드

#### 빌드 명령
```bash
cd MeshCore
pio run -e rak4631_companion
```

#### 플래싱 명령
```bash
pio run -e rak4631_companion -t upload
```

#### 시리얼 모니터
```bash
pio device monitor -b 115200
```

## 6. 설정 및 구성

### 6.1 초기 설정 (CLI)

웹 플래셔 또는 시리얼 터미널에서 설정:

```
# 노드 이름 설정
name "MyNode"

# 주파수 설정 (지역별)
# 한국: 920-923MHz (AS923)
freq 920000000

# 출력 전력 설정 (dBm)
txpower 14

# 대역폭 설정
bw 125

# 스프레딩 팩터
sf 9

# 코딩 레이트
cr 5

# 저장
save
```

### 6.2 한국 주파수 설정

| 파라미터 | 권장값 | 설명 |
|----------|--------|------|
| 주파수 대역 | 920-923 MHz | AS923 밴드 |
| 출력 전력 | 14 dBm | 법적 최대 출력 |
| 대역폭 | 125 kHz | 일반 용도 |
| SF | 7-10 | 거리/속도 트레이드오프 |

### 6.3 BLE 페어링

1. 스마트폰에서 MeshCore 앱 설치
2. 앱 실행 후 "Scan" 선택
3. 장치 목록에서 노드 선택
4. PIN 입력: `123456` (기본값)
5. 연결 완료

## 8. 문제 해결

### 8.1 일반적인 문제

| 문제 | 원인 | 해결방법 |
|------|------|----------|
| 플래싱 실패 | 부트로더 모드 아님 | BOOT 버튼 누르고 리셋 |
| BLE 연결 안됨 | 다른 장치 연결됨 | 기존 페어링 해제 |
| 메시지 전송 실패 | 주파수 불일치 | 동일 주파수 설정 확인 |
| 짧은 통신 거리 | 안테나 문제 | 안테나 연결 확인 |

### 8.2 nRF52 부트로더 모드 진입
1. RESET 버튼 2번 빠르게 누르기
2. LED가 페이딩(숨쉬기) 모드로 변경
3. 이 상태에서 플래싱 진행
