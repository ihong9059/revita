# RAK4630 모듈 검토 자료

**작성일:** 2026-03-03
**목적:** 리비타 프로젝트용 LoRa/BLE 모듈 기술 검토

---

## 1. 제품 개요

RAK4630은 RAKwireless에서 제조한 **LoRa + BLE 5.0 듀얼 무선 통신 모듈**입니다.

| 항목 | 내용 |
|------|------|
| 제조사 | RAKwireless |
| MCU | Nordic nRF52840 (ARM Cortex-M4F) |
| LoRa 칩 | Semtech SX1262 |
| 통신 방식 | LoRaWAN + BLE 5.0 |
| 제품군 | WisDuo Series |

---

## 2. 상세 사양

### 2.1 물리적 사양

| 항목 | 사양 |
|------|------|
| 크기 | **15 x 23 x 3 mm** |
| 공차 | ±0.05 mm |
| 패키지 | SMD (Stamp Module) |

### 2.2 전원 사양

| 항목 | 최소 | 일반 | 최대 | 단위 |
|------|------|------|------|------|
| VBAT_SX (LoRa) | 2.0 | 3.3 | 3.7 | V |
| VDD_NRF (MCU) | 2.0 | 3.3 | 3.6 | V |
| VBUS (USB) | 4.35 | 5.0 | 5.5 | V |

### 2.3 전류 소모

| 동작 모드 | 전류 | 조건 |
|----------|------|------|
| **Sleep (Deep Sleep)** | **4.23 uA** | LoRaWAN 슬립, 3.3V |
| Idle | 3.35 mA | 시스템 활성, 무전송 |
| TX (송신) | 67.8 mA | 15바이트, 3.6V, 1초 |
| RX Window | 2.22 mA | 3.6V, 1초 |

### 2.4 동작 환경

| 항목 | 범위 |
|------|------|
| 동작 온도 | **-40°C ~ +85°C** |
| 보관 온도 | -40°C ~ +85°C |

## 3. RF 성능

### 3.1 LoRa 사양

| 항목 | 사양 |
|------|------|
| 트랜시버 | Semtech SX1262 |
| 최대 출력 | **+22 dBm** (PA Boost 모드) |
| 대역폭 | 7.8 kHz ~ 500 kHz |
| 링크 버짓 | **최대 170 dB** |
| 프로토콜 | LoRaWAN 1.0.3 (Class A/B/C) |
| P2P 통신 | 지원 |

### 3.2 지원 주파수 대역

| 모델 | 주파수 | 대상 지역 |
|------|--------|----------|
| **RAK4630(H)** | 779-923 MHz | 유럽, 미국, 인도, 호주, 한국 |
| **RAK4630(L)** | 433-470 MHz | 중국, 433MHz 대역 |

**구체적 대역:**
- EU868, US915, AU915, AS923, KR920, IN865, RU864, CN470

### 3.3 BLE 사양

| 항목 | 사양 |
|------|------|
| 표준 | Bluetooth 5.0 (BLE) |
| TX 출력 | -20 ~ +4 dBm (4dB 단계) |
| 용도 | 설정, 근거리 제어, 데이터 전송 |

## 4. 인터페이스

| 인터페이스 | 설명 |
|-----------|------|
| UART | AT 명령어 인터페이스 (UART1) |
| USB | USB-C 지원 |
| I2C | 듀얼 포트 |
| SPI | SX1262 내부 연결 |
| GPIO | 다수의 MCU 핀 노출 |
| NFC | P0.10, P0.09 |
| SWD | 디버깅 인터페이스 |
| ADC | 아날로그 입력 지원 |

### 4.1 주요 핀 배치 (nRF52840 ↔ SX1262 내부 연결)

| nRF52840 핀 | SX1262 핀 | 기능 |
|------------|----------|------|
| P1.12 | MOSI | SPI MOSI |
| P1.13 | MISO | SPI MISO |
| P1.14 | BUSY | BUSY 신호 |
| P1.15 | DIO1 | 인터럽트 |
| P1.06 | NRESET | SX1262 리셋 |

## 5. LoRa Mesh 지원 검토

### 5.1 Meshtastic 지원 현황

| 항목 | 지원 여부 | 비고 |
|------|----------|------|
| **Meshtastic 펌웨어** | **지원** | 공식 지원 디바이스 |
| LoRa Mesh 네트워크 | **지원** | Meshtastic 프로토콜 기반 |
| P2P 통신 | 지원 | LoRa Point-to-Point |
| 커스텀 Mesh | 가능 | 펌웨어 커스터마이징 필요 |

### 5.2 Meshtastic 관련 제품군

RAKwireless는 RAK4630/RAK4631 기반 Meshtastic 전용 제품을 출시하고 있습니다:

1. **WisBlock Meshtastic Starter Kit**
   - RAK4631 + WisBlock Base
   - Meshtastic 펌웨어 프리로드

2. **WisMesh Pocket**
   - 휴대용 Meshtastic 디바이스
   - 조립 불필요, 즉시 사용 가능

3. **WisMesh Repeater**
   - 고정형 실외 배치용
   - IP67 등급, 태양광 패널 지원
   - 장기간 원격 운영 가능

### 5.3 Meshtastic 펌웨어 설치

```
펌웨어 파일: firmware-rak4631-w.x.yy.zzzzzzz.uf2
설치 방법: flasher.meshtastic.org → WisBlock RAK4631 선택 → UF2 다운로드 → USB-C로 복사
```

### 5.4 Mesh 네트워크 성능 (사용자 보고)

| 항목 | 수치 |
|------|------|
| 전력 소모 | **5mA 미만** (T-Beam 대비 우수) |
| 메시 홉 | 다중 홉 지원 |
| 커버리지 | 최대 수 km (환경에 따라 다름) |

## 6. 개발 환경

### 6.1 지원 플랫폼

| 플랫폼 | 설명 |
|--------|------|
| **RUI3** | RAKwireless 기본 펌웨어 (AT 명령어 지원) |

| **nRF Connect SDK** | Nordic 공식 SDK |
| **Zephyr RTOS** | 오픈소스 RTOS |
| **Meshtastic** | LoRa Mesh 펌웨어 |

### 6.2 개발 자유도

- **오픈 플랫폼**: 제한 없이 커스텀 펌웨어 플래싱 가능
- **AT 명령어**: USB, UART, BLE를 통한 외부 MCU 연동

## 9. 프로젝트 적합성 평가

### 9.1 장점

| 항목 | 평가 |
|------|------|
| 듀얼 통신 | LoRa + BLE 동시 지원으로 유연한 설계 가능 |
| 저전력 | Sleep 4.23uA로 배터리 구동에 적합 |
| Mesh 지원 | Meshtastic으로 LoRa Mesh 즉시 구현 가능 |
| 개발 용이성 | Arduino, RUI3 등 다양한 개발 환경 |
| 소형 크기 | 15x23mm로 소형 제품 설계 가능 |
| 인증 완료 | KC 포함 글로벌 인증 취득 |
| 넓은 온도 범위 | -40~+85°C 산업용 등급 |

### 9.2 고려사항

| 항목 | 내용 |
|------|------|
| 주파수 선택 | 한국은 KR920 대역 사용 (RAK4630(H) 필요) |
| 안테나 | 외부 안테나 필요 (50옴 임피던스 매칭) |
| RF 설계 | 노이즈 소스와 거리 유지 필요 |
| Mesh 제한 | 기본 LoRaWAN은 Star 토폴로지, Mesh는 Meshtastic 필요 |

### 9.3 결론

**RAK4630은 리비타 프로젝트에 적합합니다.**

- LoRa Mesh 필요 시: **Meshtastic 펌웨어로 지원 가능**
- 저전력 IoT 노드 구현에 최적화
- 한국 KC 인증 완료로 국내 사용 가능
- 다양한 개발 환경 지원으로 빠른 프로토타이핑 가능
