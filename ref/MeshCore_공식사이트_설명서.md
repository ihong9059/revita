# MeshCore 공식 사이트 설명서

> **URL**: https://meshcore.co.uk/

## 1. MeshCore 개요

MeshCore는 **LoRa 라디오 하드웨어를 활용한 보안 텍스트 기반 통신을 가능하게 하는 멀티 플랫폼 시스템**입니다.

### 핵심 미션
신뢰성, 보안, 효율성에 중점을 둔 **분산형 메시 라디오 네트워킹 시스템** 개발

### 핵심 콘셉트
**"Off-line and Off-grid messaging platform"**
- 인터넷 없이 사람과 사물을 연결
- 완전 독립적인 통신 네트워크 구축

---

## 2. 주요 용도 및 활용 분야

| 분야 | 설명 |
|------|------|
| **오프그리드 통신** | 인터넷/셀룰러 네트워크가 없는 지역에서의 통신 |
| **긴급 대응 및 재난 복구** | 재난 상황에서 인프라 독립적인 통신망 구축 |
| **야외 활동** | 하이킹, 캠핑, 등산 등 야외 활동 시 통신 |
| **전술 보안** | 법집행, 민간 보안 등 보안이 필요한 통신 |
| **IoT 센서 네트워크** | 원격 센서 데이터 수집 및 모니터링 |

---

## 3. 주요 특징

### 3.1 완전 분산형 아키텍처
- 중앙 서버 불필요
- 자가 치유(Self-healing) 네트워크 구조
- 단일 장애점(Single Point of Failure) 없음

### 3.2 LoRa 기반 장거리 통신
- 저전력 장거리 통신 프로토콜 사용
- 수 km 이상의 통신 거리 지원
- 다양한 LoRa 하드웨어 호환

### 3.3 보안 통신
- 암호화된 메시지 전송
- 개인키 기반 인증
- 오프라인에서도 보안 유지

### 3.4 멀티홉 라우팅
- 중간 노드를 통한 메시지 릴레이
- 네트워크 범위 자동 확장
- 동적 경로 설정
