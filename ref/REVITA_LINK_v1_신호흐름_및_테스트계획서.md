│             │     │  Si2302CDS  │     │             │     │             │

                    │   12V_SW    │
                    │  (12V_VDD)  │
                    └─────────────┘

[동작 설명]
- EN_P2 = HIGH: 전체 전원 ON (GND 연결)
- EN_A = HIGH: A 방향으로 전류 흐름 (M3 ON)
- EN_B = HIGH: B 방향으로 전류 흐름 (M4 ON)
- EN_A, EN_B 동시 HIGH 금지 (단락 위험)

[TO_GND 의미]
- N-Channel MOSFET Si2302CDS의 드레인 출력
- EN_P2 = HIGH 시 LOAD_IN을 GND로 연결
- 모터의 공통 GND 스위칭 역할

### 3.5 레벨 변환 (12V DIO → 3.3V DIO)

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  12V_DIO_IN │────►│   PC817C    │────►│ 3.3V_DIO_OUT│
│ (외부 12V)  │     │ (Optocoupler)│    │ (MCU 입력)  │
└─────────────┘     └─────────────┘     └─────────────┘

[동작 원리]
- 12V_DIO_IN = HIGH (12V)
  → LED ON → 포토트랜지스터 ON
  → 3.3V_DIO_OUT = LOW (Active Low)

- 12V_DIO_IN = LOW (0V)
  → LED OFF → 포토트랜지스터 OFF
  → 3.3V_DIO_OUT = HIGH (풀업)

[절연 이유]
- 12V 외부 신호와 3.3V MCU 간 전기적 절연
- 서지/노이즈로부터 MCU 보호
- GND 분리 (GND_IN vs GND)
```

### 3.6 3.0V Watchdog (TPS3808G30DBVR)

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│    VBAT     │────►│  TPS3808    │────►│   RESET#    │────► RAK4630 NRF_RESET
│ (Battery)   │     │ (Watchdog)  │     │ (Active Low)│
└─────────────┘     └─────────────┘     └─────────────┘

[Watchdog 사용 이유]
┌──────────────────────────────────────────────────────────────────────┐
│ 1. 저전압 보호 (UVLO)                                                │
│    - VBAT < 3.0V 시 RESET# = LOW 출력                               │
│    - 배터리 방전 시 MCU를 리셋 상태로 유지                           │
│    - MCU가 불안정한 전압에서 오작동하는 것을 방지                    │
│                                                                      │
│ 2. 파워온 리셋 (POR)                                                 │
│    - 전원 인가 시 일정 시간 후 RESET# 해제                           │
│    - 전원이 안정화된 후 MCU 시작 보장                                │
│                                                                      │
│ 3. CT 핀 (Delay 설정)                                                │
│    - C2 (1nF) 연결로 리셋 지연 시간 설정                             │
│    - 노이즈에 의한 오동작 방지                                       │
└──────────────────────────────────────────────────────────────────────┘
```

### 4.4 문제 발생 시 디버깅 가이드

| 문제 | 가능한 원인 | 해결 방법 |
|------|-------------|-----------|
| 3.3V 출력 안됨 | TPS63001 솔더링 불량, VIN 미인가 | 솔더링 확인, VIN 전압 확인 |
| MCU 인식 안됨 | SWD 연결 불량, 전원 문제 | 연결선 확인, 전원 확인 |
| I2C 응답 없음 | 주소 오류, 풀업 저항 누락 | 주소 확인 (0x20), 풀업 확인 |
| RS485 통신 안됨 | DE/RE 제어 오류, A/B 바뀜 | 송수신 모드 확인, A/B 극성 확인 |
| 모터 동작 안함 | 12V 미출력, EN 신호 오류 | 12V_EN 확인, EN 신호 순서 확인 |
| 12V 출력 낮음 | 부하 과다, 인덕터 포화 | 부하 감소, 인덕터 확인 |
