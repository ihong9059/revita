## 1. 개요

### 1.1 프로젝트 목적
RAK4630 모듈을 활용하여 인터넷 독립적인 LoRa Mesh 네트워크를 구축하고, MeshCore 펌웨어를 적용하여 오프그리드 통신 시스템을 개발한다.

### 1.2 RAK4630 모듈 개요
RAK4630은 Nordic nRF52840 MCU와 Semtech SX1262 LoRa 트랜시버를 통합한 저전력 장거리 무선 통신 모듈이다.

### 4.3 규제 및 인증 문제

#### 4.3.1 한국 전파법
| 항목 | 요구사항 | 현황 |
|------|----------|------|
| 주파수 대역 | KR920 (920-923 MHz) | 지원 (RAK4630(H)) |
| 최대 출력 | 14 dBm (25mW) | 소프트웨어 제한 필요 |
| Duty Cycle | 제한 있음 | 펌웨어에서 관리 |
| 형식 승인 | KC 인증 필요 | 양산 시 필요 |

#### 4.3.2 주의사항
```
중요: 한국에서 LoRa 사용 시 최대 출력을 14dBm으로 제한해야 함
현재 MeshCore 설정: 22dBm (해외용)
수정 필요: LORA_TX_POWER=14
```

### 7.2 권고사항

#### 즉시 조치 필요
1. **출력 제한**: `LORA_TX_POWER=14` 로 변경 (한국 법규)
2. **모듈 구매**: RAK4630(H) - KR920 지원 버전 확인

#### 개발 시 주의사항
1. SWD 디버깅 환경 구축 권장
2. 충분한 야외 테스트 수행
3. 배터리 수명 실측 필요
4. 안테나 성능이 전체 시스템 성능 좌우

1. KC 인증 취득 필요
2. EMC 테스트 수행
3. 신뢰성 테스트 (온도, 진동, 수명)
4. 품질 관리 체계 구축
