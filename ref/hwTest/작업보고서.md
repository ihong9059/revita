## 중요 발견사항 (반복 실수 방지)

### Bare-metal GPIO 설정

nRF52840 bare-metal 코드에서 UART가 작동하지 않을 때:

```c
// 잘못된 설정 (작동 안함)
P0_PIN_CNF(TX_PIN) = 0x00000001;
P0_PIN_CNF(RX_PIN) = 0x00000000;

// 올바른 설정 (작동함)
P0_PIN_CNF(TX_PIN) = 0x00000003;  // DIR=Output, INPUT=Disconnect
P0_PIN_CNF(RX_PIN) = 0x0000000C;  // DIR=Input, PULL=Pullup
```

**검증 방법**: Zephyr serial_counter로 하드웨어 확인 후, bare-metal 코드 디버깅
