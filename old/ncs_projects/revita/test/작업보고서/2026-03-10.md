### 2. 펌웨어 기능 추가

#### 2.1 UART Echo 기능 구현

- **요구사항**: 시리얼 입력을 받으면 `resend: ` 를 붙여서 반환
- **구현 방법**:
  - `console_getchar()` 사용하여 문자 수신
  - 버퍼에 저장 후 Enter 입력 시 `resend: ` prefix 붙여서 출력

#### 2.2 멀티스레드 구조로 변경

- **LED Blink Thread**: 100ms 주기로 LED 점멸 (10배 빠르게) + 카운터 출력
- **Main Thread**: UART 입력 처리 및 echo

#### 2.3 UART1 추가 (신규)

- **요구사항**: 두 번째 UART로 카운터 출력
- **핀 배치**:
  - TX: P1.01
  - RX: P1.02
- **출력 형식**: `UART2 Counter: N`

### 7. 테스트 방법

**UART0 (Console - /dev/ttyACM0):**
```bash
minicom -D /dev/ttyACM0 -b 115200
```

**UART1 (P1.01 TX) - USB-Serial 어댑터 필요:**
```bash
minicom -D /dev/ttyUSB0 -b 115200
```

**예상 출력 (UART0):**
```
UART Test Started
UART0: Console + Echo
UART1 (P1.01 TX, P1.02 RX): Counter output
Type something and press Enter:
Counter: 1
Counter: 2
hello        <- 입력
resend: hello
Counter: 3
...
```

UART2 Counter: 2

...
