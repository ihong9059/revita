- **목적**: Dual UART 출력 테스트
- **기능**: 1초마다 카운터 값을 UART1, UART2로 동시 출력

| UART1 (uart1) | 카운터 출력 | P0.20 | P0.19 | 115200 |

### 5. 주요 파일 내용

#### 5.1 prj.conf

```
CONFIG_SERIAL=y
CONFIG_UART_INTERRUPT_DRIVEN=y
CONFIG_PRINTK=y
```

#### 5.2 app.overlay (핀 설정)

```dts
/* UART0 (UART2): TX P0.16, RX P0.15 */
/* UART1: TX P0.20, RX P0.19 */

&pinctrl {
    uart0_default: uart0_default {
        group1 { psels = <NRF_PSEL(UART_TX, 0, 16)>; };
        group2 { psels = <NRF_PSEL(UART_RX, 0, 15)>; bias-pull-up; };
    };
    uart1_default: uart1_default {
        group1 { psels = <NRF_PSEL(UART_TX, 0, 20)>; };
        group2 { psels = <NRF_PSEL(UART_RX, 0, 19)>; bias-pull-up; };
    };
};
```

#### 5.3 main.c 핵심 로직

```c
while (1) {
    snprintf(buf, sizeof(buf), "UART2 count: %u\r\n", counter);
    uart_send_string(uart2_dev, buf);

    snprintf(buf, sizeof(buf), "UART1 count: %u\r\n", counter);
    uart_send_string(uart1_dev, buf);

    counter++;
    k_sleep(K_SECONDS(1));
}
```

### 6. 빌드 및 플래시

```bash
# 환경 설정
source ~/ncs/.venv/bin/activate

# 빌드
cd ~/ncs/revita/first
west build -b nrf52840dk/nrf52840 . --pristine

# 플래시 (target1 - S/N 1050234191)
JLinkExe -device nRF52840_xxAA -if SWD -speed 4000 \
  -SelectEmuBySN 1050234191 \
  -CommandFile /tmp/jlink_flash.jlink

# 플래시 (target2 - S/N 1050295470)
JLinkExe -device nRF52840_xxAA -if SWD -speed 4000 \
  -SelectEmuBySN 1050295470 \
  -CommandFile /tmp/jlink_flash.jlink
```

**jlink_flash.jlink 내용:**
```
device nRF52840_xxAA
si SWD
speed 4000
connect
erase
loadfile /home/uttec/ncs/revita/first/build/first/zephyr/zephyr.hex
r
g
exit
```
